# MFSD Assignments - MERN Full Stack Development

## Folder Structure

| Folder | Assignment | Type | How to Run |
|--------|-----------|------|-----------|
| 01_login_form | Login Form with Validation | HTML/CSS/JS | Open index.html in browser |
| 02_todo_list | To-Do List App | HTML/CSS/JS | Open index.html in browser |
| 03_student_list | Student List Manager | HTML/CSS/JS | Open index.html in browser |
| 04_profile_card | User Profile Card | React | npm install → npm run dev |
| 05_props_demo | Props Parent to Child | React | npm install → npm run dev |
| 06_student_form | Student Form with useState | React | npm install → npm run dev |
| 07_useeffect_fetch | Fetch API with useEffect | React | npm install → npm run dev |
| 08_form_validation | Form Validation in React | React | npm install → npm run dev |
| 09_useeffect_lifecycle | useEffect Lifecycle Demo | React | npm install → npm run dev |
| 10_express_crud | Express CRUD REST API | Express | npm install → npm run dev |
| 11_express_post | Express POST Form Data | Express | npm install → npm run dev |
| 12_express_middleware | Express Logger Middleware | Express | npm install → npm run dev |
| 13_express_put_delete | Express PUT & DELETE | Express | npm install → npm run dev |
| 14_mongoose_user | Mongoose User Schema | Express + MongoDB | npm install → npm run dev |
| 15_mongoose_product | Mongoose Product Schema | Express + MongoDB | npm install → npm run dev |

## Quick Tips
- Assignments 1-3: Just open the HTML file in browser (no install needed)
- Assignments 4-9: Need Node.js installed. Run `npm install` then `npm run dev`
- Assignments 10-13: Need Node.js. Run `npm install` then `npm run dev`, test via Postman
- Assignments 14-15: Need Node.js + MongoDB running locally (`mongod` command)
